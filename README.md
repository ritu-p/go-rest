# Go REST Boilerplate


Simple starter using Gin + GORM (Postgres).


## Run with Docker Compose


docker-compose up --build


## Run locally


1. copy .env
2. go run ./cmd/server


Endpoints:
- POST /api/v1/users -> create user
- GET /api/v1/users -> list users
- GET /api/v1/users/:id -> get user by id
